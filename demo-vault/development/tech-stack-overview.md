# Technical Stack Overview

## Frontend
- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State:** Zustand

## Backend
- **Database:** Supabase (PostgreSQL)
- **Vector Search:** pgvector
- **API:** Next.js API Routes

## AI/ML
- **Embeddings:** OpenAI ada-002
- **Chat:** OpenRouter (Claude 3.5)
