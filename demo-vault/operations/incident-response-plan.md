# Incident Response Plan

## Severity Levels

### P0: Critical
- App down
- Data breach
- Response: Immediate

### P1: High
- Feature broken
- Performance degraded
- Response: <2 hours

### P2: Medium
- Minor bugs
- Response: <1 day
