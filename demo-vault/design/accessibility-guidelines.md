# Accessibility Guidelines

## WCAG 2.1 AA Compliance

### Color Contrast
- Text: 4.5:1 minimum
- Large text: 3:1 minimum
- UI components: 3:1 minimum

### Keyboard Navigation
- All features accessible via keyboard
- Focus indicators visible
- Skip links provided
